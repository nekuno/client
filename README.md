# Nekuno hybrid mobile app

## Install

`git clone https://github.com/nekuno/client.git`

`cd client`

`npm install`

## Compile with webpack:

For development:

`npm run build`

For production:

`npm run build:prod`

## Testing

`npm run test`

## Stories (ui development)

`npm run storybook`

and go to [localhost:9001]()
